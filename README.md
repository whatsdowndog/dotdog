# whatsdown.dog

## Development

1. Install [Zola](https://www.getzola.org/documentation/getting-started/installation/)
1. Run `zola serve`
1. Open up [http://127.0.0.1:1111](http://127.0.0.1:1111) (or the port it says)
1. Edit away!

For more help, see the [Zola docs](https://www.getzola.org/documentation/getting-started/overview/) or the [Tera docs](https://keats.github.io/tera/docs/).
